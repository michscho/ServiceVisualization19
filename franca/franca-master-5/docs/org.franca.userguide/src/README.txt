--------------------------------------------------------------------------------
FRANCA DOCUMENTATION STRUCTURE
--------------------------------------------------------------------------------

The Franca documentation is available as pdf (via LaTeX), html and
Eclipse online help. All these formats are generated from a common
source using Xdoc. Franca is using a special fork of Xdoc, called
Xdoc4Franca. Xdoc4Franca is available on github.com [1].

--------------------------------------------------------------------------------

The overall Franca documentation is structured into four parts:

- Part I:   Introductory material (Getting started, concepts overview, tools overview) 
- Part II:  Reference material (language references for IDL and deployment DSL)
- Part III: Integration with other tools and projects (e.g., D-Bus, CommonAPI)
- Part IV:  For developers (model API, building extensions, generators, etc.) 

--------------------------------------------------------------------------------

[1] https://github.com/kbirken/xDoc
