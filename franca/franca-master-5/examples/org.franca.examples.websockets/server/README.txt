--------------------------------------------------------------------------------------------------

This example Websocket server is implemented in JavaScript based on node.js and websocket.io.

--------------------------------------------------------------------------------------------------

INSTALLATION:
1. Install node.js: Visit http://nodejs.org and click on the Install button to get started.
2. Install node.js modules by "npm install" (will install mime and websocket.io).

--------------------------------------------------------------------------------------------------

HOW TO RUN IT:
1. Generate the stub's JS code by executing the WebsocketGenTest JUnit test in the Eclipse IDE.
   You need Franca 0.12.0 or later for doing this successfully.
2. In this folder, simply issue "node index.js". This will start the server on port 8080.
3. Open your browser and browse to "localhost:8080" in order to start the client side.
   We have tested this with the Google Chrome (version 59.0.3071.115), and Mozilla Firefox.
